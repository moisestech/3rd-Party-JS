### **Add CSS to console messages**

- Add CSS to console messages

  ```js
  console.log("%cColor", "color: blue; font-size: x-large");
  ```

- Want to color only a specific word from the log message?

  ```js
  console.log("You just %c Won! a lottry", "background: #222; color: #bada55");
  ```
